# Jimmy Profile

一个以猫咪 Jimmy（吉咪）为主角的互动数字名片页。

页面采用便签式排版与温暖纸张风格，中心是会“回望”你的猫咪头像：在桌面端，Jimmy 会跟随鼠标方向转头；在移动端，可以通过触摸互动，并在支持的设备上开启重力感应模式。

## 功能简介

- 桌面端鼠标视线跟随
- 移动端触摸跟随
- 支持设备方向感应的移动端互动
- 基于 sprite sheet 的多帧猫咪转头动画
- 响应式卡片布局

## 项目结构

```text
src/
  index.html       页面结构
  styles.css       页面视觉与响应式样式
  main.js          资料填充、视线跟随与感应逻辑
  assets/
    sprite.webp    猫咪多帧精灵图
```

## 本地预览

这是一个纯静态项目，不依赖构建工具。进入 `src` 目录后，用任意静态服务器启动即可。

## 交互说明

- 桌面端：移动鼠标，Jimmy 会根据方向切换对应帧。
- 移动端：拖动或按住头像区域可触发触摸互动。
- 支持陀螺仪的设备：可点击“开启感应模式”启用方向感应。
